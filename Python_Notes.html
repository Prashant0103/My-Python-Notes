Introduction to Python

Python is an open-source, high-level programming language that is known for its ease of learning and use. It is one of the most popular languages in the world, widely adopted in various fields such as web development, data science, machine learning, artificial intelligence (AI), automation, and more. Its syntax is simple and clear, allowing developers to focus on solving problems rather than getting caught up in complex syntax rules.

  • Cross-platform: Python is available on all major platforms, including Windows, macOS, and Linux. This means you can write Python code on one platform and run it on others without modification, making it ideal for cross-platform development.
  • Supports multiple programming paradigms: Python is a multi-paradigm language, meaning it supports object-oriented, procedural, and functional programming. This flexibility allows developers to choose the best approach for the problem they are solving.
  • Ideal for web development, data analysis, AI, and more: Python is highly versatile. It powers popular web development frameworks like Django and Flask, data analysis libraries like Pandas and NumPy, and machine learning libraries such as TensorFlow and Scikit-learn. This makes Python an excellent choice for building a wide variety of applications—from websites and data-driven solutions to intelligent systems.

Python Syntax

Python uses indentation to define the structure of the code, making it cleaner and more readable compared to other programming languages that rely on braces or other symbols to define code blocks. This approach enforces good coding practices, as the indentation level directly represents the block of code. Python’s syntax is designed to be simple and intuitive, making it an excellent language for beginners as well as experienced developers.

Python statements do not require semicolons to mark the end of a line, and code blocks (such as the body of a function or a loop) are defined by consistent indentation. Proper indentation is crucial in Python; otherwise, the interpreter will raise an indentation error. 

print("Hello, World!")

Comments

Comments in Python are lines of text in the code that are not executed by the interpreter. They are used to explain the code, making it more understandable for others (or even for yourself when you revisit the code later). Comments can describe the logic behind the code, why certain decisions were made, or even leave TODO notes for future work.

In Python, comments are created by placing a # symbol in front of the text. Anything following the # on the same line is considered a comment and will be ignored by the Python interpreter. 

# This is a comment

Variables

Variables in Python are used to store data that can be referenced and manipulated in your program. Unlike many other programming languages, Python is dynamically typed, meaning you don’t need to explicitly declare the type of a variable. Python automatically determines the type based on the value assigned to the variable.

A variable can hold various types of data, such as integers, strings, lists, and more. Once a variable is created, you can reassign it to store a different value. This dynamic nature of Python makes it very flexible. 

x = 5
    y = "Hello"
    print(x)
    print(y)

Data Types

Python has various built-in data types:

  • Numeric: Used to store numeric values.
    • int: Integer numbers (e.g., x = 10)
    • float: Floating-point numbers (e.g., y = 10.5)
    • complex: Complex numbers (e.g., z = 3 + 5j)
  • Text: Used to store text data.
    • str: String of characters (e.g., name = "Alice")
  • Sequence: Used to store multiple items in a single variable.
    • list: Ordered, mutable collection (e.g., fruits = ["apple", "banana"])
    • tuple: Ordered, immutable collection (e.g., colors = ("red", "green"))
    • range: Sequence of numbers (e.g., numbers = range(5))
  • Dictionary: Stores key-value pairs.
    • dict: Dictionary (e.g., person = {"name": "Alice", "age": 25})
  • Set: Collection of unordered unique items.
    • set: Mutable set (e.g., items = {"apple", "banana"})
    • frozenset: Immutable set (e.g., fset = frozenset(["apple", "banana"]))
  • Boolean: Represents truth values.
    • bool: True or False (e.g., is_active = True)
  • Binary: Used to store binary data.
    • bytes: Immutable binary data (e.g., b = b"hello")
    • bytearray: Mutable binary data (e.g., ba = bytearray([65, 66]))

Operators

Python supports various operators:

  • Arithmetic Operators: Perform mathematical operations.
    • +: Addition (3 + 2 = 5)
    • -: Subtraction (5 - 2 = 3)
    • *: Multiplication (4 * 2 = 8)
    • /: Division (10 / 2 = 5.0)
    • %: Modulus (remainder) (10 % 3 = 1)
    • **: Exponentiation (2 ** 3 = 8)
    • //: Floor division (10 // 3 = 3)
  • Assignment Operators: Assign and update values.
    • =: Assign value (x = 5)
    • +=: Add and assign (x += 3 is x = x + 3)
    • -=: Subtract and assign (x -= 2)
    • *=: Multiply and assign (x *= 4)
    • /=: Divide and assign (x /= 2)
    • %=: Modulus and assign (x %= 3)
    • **=: Exponent and assign (x **= 2)
    • //=: Floor divide and assign (x //= 2)
  • Comparison Operators: Compare values.
    • ==: Equal to (5 == 5 → True)
    • !=: Not equal to (5 != 3 → True)
    • >: Greater than (5 > 3 → True)
    • <: Less than (3 < 5 → True)
    • >=: Greater than or equal to (5 >= 5 → True)
    • <=: Less than or equal to (3 <= 5 → True)
  • Logical Operators: Combine boolean expressions.
    • and: True if both are true (True and False → False)
    • or: True if at least one is true (True or False → True)
    • not: Inverts the result (not True → False)
  • Identity Operators: Compare memory locations.
    • is: True if same object (x is y)
    • is not: True if not same object (x is not y)
  • Membership Operators: Test membership in a sequence.
    • in: True if value exists ("a" in "apple" → True)
    • not in: True if value doesn't exist ("x" not in "apple" → True)
  • Bitwise Operators: Operate on binary representations.
    • &: AND (5 & 3 = 1)
    • |: OR (5 | 3 = 7)
    • ^: XOR (5 ^ 3 = 6)
    • ~: NOT (~5 = -6)
    • <<: Left shift (5 << 1 = 10)
    • >>: Right shift (5 >> 1 = 2)

Lists

A List in Python is an ordered collection of items that is mutable, meaning you can change, add, or remove elements after the list has been created. Lists can hold items of any data type, including a mix of types. 

# Creating a list of fruits
    fruits = ["apple", "banana", "cherry"]
    
    # Accessing the first item in the list using index 0
    print(fruits[0])  # Output: apple

Tuples

A Tuple in Python is an ordered collection of items that is immutable, meaning its elements cannot be changed, added, or removed after creation. Tuples are commonly used for fixed data and are often faster than lists for iteration. 

# Creating a tuple of fruits
    fruits = ("apple", "banana", "cherry")
    
    # Accessing the first item in the tuple using index 0
    print(fruits[0])  # Output: apple

Sets

A Set in Python is an unordered collection of unique items. Sets do not allow duplicate values and are commonly used for membership testing, removing duplicates, and performing set operations like union and intersection. 

# Creating a set of fruits
    fruits = {"apple", "banana", "cherry"}
    
    # Printing the set (order may vary, and duplicates are automatically removed)
    print(fruits)  # Output: {'banana', 'apple', 'cherry'} (order is not guaranteed)

Conditions

Python supports conditional statements that allow you to execute code blocks based on specific conditions. These include if, elif (else if), and else clauses to handle multiple scenarios. 

# Defining a variable
    x = 10
    
    # Checking if x is greater than 5
    if x > 5:
        print("x is greater than 5")
    
    # Checking if x is equal to 5 (only checked if the previous condition is False)
    elif x == 5:
        print("x is 5")
    
    # If none of the above conditions are true
    else:
        print("x is less than 5")

Loops

Python supports two main types of loops: for loops and while loops. Loops are used to execute a block of code repeatedly, either for a fixed number of times or while a condition is true. 

# For loop: iterates over a sequence (like a range or list)
    for i in range(5):
        print(i)  # Output: 0 1 2 3 4
    
    # While loop: runs as long as the condition is true
    i = 0
    while i < 5:
        print(i)  # Output: 0 1 2 3 4
        i += 1  # Increment i to avoid infinite loop

Functions

A Function in Python is a reusable block of code that performs a specific task. Functions help organize code into manageable sections and are defined using the def keyword, followed by the function name and parameters. 

# Defining a function named 'greet' that takes one parameter 'name'
    def greet(name):
        print("Hello, " + name)
    
    # Calling the function and passing the argument "Alice"
    greet("Alice")  # Output: Hello, Alice

Lambda

A lambda function is a short, one-line function with no name. It's used when you need a simple function for a short time. You can use it instead of writing a full def function. 

# Creating a lambda function that adds two numbers
    add = lambda a, b: a + b
    
    # Calling the lambda function
    print(add(5, 3))  # Output: 8

Arrays

An array is a special type of list used to store only one type of data (usually numbers). It is more memory-efficient than regular lists and is useful when you work with large amounts of numerical data. You need to import the array module to use arrays in Python. 

# Importing the array module
    import array
    
    # Creating an integer array ('i' stands for integer)
    arr = array.array('i', [1, 2, 3, 4, 5])
    
    # Printing the array
    print(arr)  # Output: array('i', [1, 2, 3, 4, 5])

Classes/Objects

Python supports object-oriented programming (OOP), which allows you to create classes and objects. A class is a blueprint for creating objects, and an object is an instance of a class. Classes can have attributes (variables) and methods (functions). 

# Defining a class named 'Person'
    class Person:
        # Constructor method to initialize the object with a name
        def __init__(self, name):
            self.name = name
    
        # Method to print a greeting message
        def greet(self):
            print("Hi, I'm " + self.name)
    
    # Creating an object of the Person class
    p = Person("Bob")
    
    # Calling the greet method of the object
    p.greet()  # Output: Hi, I'm Bob

Inheritance

Inheritance allows a class to inherit attributes and methods from another class. This helps create a new class based on an existing class, enabling code reuse and creating a hierarchical relationship between classes. 

# Defining a base class 'Person'
    class Person:
        def __init__(self, name):
            self.name = name
    
        def greet(self):
            print("Hi, I'm " + self.name)
    
    # Defining a subclass 'Student' that inherits from 'Person'
    class Student(Person):
        def __init__(self, name, student_id):
            # Calling the constructor of the parent class (Person)
            super().__init__(name)
            # Adding a new attribute for student ID
            self.student_id = student_id
    
    # Creating an object of the 'Student' class
    s = Student("Alice", "S12345")
    
    # Calling the greet method inherited from 'Person'
    s.greet()  # Output: Hi, I'm Alice
    print(s.student_id)  # Output: S12345

Polymorphism

Polymorphism allows different classes to define the same method, but with different behavior. This enables objects of different classes to be treated as objects of a common superclass and allows methods to behave differently depending on the object type. 

# Defining a class 'Dog' with a 'sound' method
    class Dog:
        def sound(self):
            return "Woof"
    
    # Defining a class 'Cat' with a 'sound' method
    class Cat:
        def sound(self):
            return "Meow"
    
    # Creating a list of animals (Dog and Cat objects)
    animals = [Dog(), Cat()]
    
    # Iterating through the animals list and calling the sound method for each
    for animal in animals:
        print(animal.sound())  # Output: Woof  Meow

Iterators

An iterator is an object that allows you to iterate over a sequence (like a list or tuple) one item at a time. Iterators implement two key methods: __iter__() to return the iterator object and __next__() to return the next item in the sequence. 

# Creating a tuple
    mytuple = ("apple", "banana", "cherry")
    
    # Creating an iterator from the tuple
    myit = iter(mytuple)
    
    # Accessing items using the 'next' function
    print(next(myit))  # Output: apple
    print(next(myit))  # Output: banana

Modules

Modules in Python allow you to organize your code across multiple files. A module is simply a file containing Python definitions and statements. You can import and use code from one module in another to keep your code clean and manageable. 

# mymodule.py
    # Defining a function in a module
    def greet():
      print("Hi")
    
    # main.py
    # Importing the 'mymodule' file to use its functions
    import mymodule
    
    # Calling the 'greet' function from the 'mymodule'
    mymodule.greet()  # Output: Hi

PIP

PIP is the Python package installer. It allows you to easily install and manage external libraries and packages in your Python environment. With PIP, you can install, upgrade, or uninstall libraries and manage dependencies in your projects. 

# Installing the 'numpy' package using PIP
    pip install numpy

DateTime

The datetime module in Python allows you to work with dates and times. It provides classes for manipulating dates and times in both simple and complex ways. You can use it to get the current date and time, perform date arithmetic, and format dates and times. 

# Importing the datetime module
    import datetime
    
    # Getting the current date and time
    now = datetime.datetime.now()
    
    # Printing the current date and time
    print(now)  # Output: 2025-05-08 10:00:00.123456

Math Module

The math module in Python provides a wide range of mathematical functions, such as basic arithmetic, trigonometry, logarithms, and more. It allows you to perform mathematical operations with greater efficiency. 

# Importing the math module
    import math
    
    # Calculating the square root of 16
    print(math.sqrt(16))  # Output: 4.0
    
    # Calculating 2 raised to the power of 3
    print(math.pow(2, 3))  # Output: 8.0

JSON

JSON (JavaScript Object Notation) is a lightweight, text-based format used to store and exchange data. It is widely used for data interchange between servers and web applications. In Python, the json module allows you to work with JSON data by converting it to and from Python objects. 

# Importing the json module
    import json
    
    # A JSON string containing some data
    data = '{"name": "John", "age": 30}'
    
    # Converting the JSON string to a Python dictionary
    person = json.loads(data)
    
    # Accessing the 'name' value from the converted dictionary
    print(person["name"])  # Output: John

Regular Expressions (RegEx)

Regular Expressions (RegEx) are powerful tools for pattern matching in strings. They allow you to search, match, and manipulate text based on specific patterns. In Python, the re module provides functions to work with regular expressions. 

# Importing the re module
    import re
    
    # Searching for the pattern "world" in the string "Hello world"
    pattern = re.search("world", "Hello world")
    
    # Printing the match object
    print(pattern)  # Output:

File Handling

Python provides built-in functions for working with files. You can read from, write to, and append to files. You can use the open() function to access a file, and then use methods like read() or write() to interact with its contents. 

# Reading from a file
    f = open("file.txt", "r")  # Open the file in read mode
    print(f.read())  # Read and print the content of the file
    f.close()  # Close the file after operation is done

Exception Handling

Exception Handling in Python is used to catch and handle errors in a controlled way, preventing crashes and enabling smoother program execution. The try block is used to test code that may cause an exception, the except block handles the exception, and the finally block executes code regardless of whether an error occurred. 

# Example of exception handling
    try:
      # Attempting to print an undefined variable
      print(x)
    except NameError:
      # Handling the exception if 'x' is not defined
      print("Variable is not defined")
    finally:
      # Code in the finally block will run regardless of an error
      print("This runs regardless of error")

MySQL

Python allows interaction with MySQL databases using libraries like mysql-connector. This allows you to connect to MySQL, execute queries, and retrieve data from the database directly from your Python programs. 

# Importing the mysql.connector library

NumPy

NumPy is a powerful library in Python used for numerical operations. It provides support for arrays, matrices, and a large collection of high-level mathematical functions to operate on these data structures. NumPy is widely used for scientific computing, data analysis, and machine learning. 

# Importing the numpy library with alias 'np'
    import numpy as np
    
    # Creating a NumPy array from a Python list
    arr = np.array([1, 2, 3, 4])
    
    # Printing the NumPy array
    print(arr)  # Output: [1 2 3 4]

Pandas

Pandas is a powerful data analysis library in Python that provides easy-to-use data structures like DataFrame and Series. It is widely used for handling and analyzing large datasets, with built-in functions for reading, filtering, and processing data. 

# Importing the pandas library with alias 'pd'
    import pandas as pd
    
    # Creating a dictionary with sample data
    data = {'Name': ['Tom', 'Jerry'], 'Age': [20, 21]}
    
    # Converting the dictionary into a DataFrame
    df = pd.DataFrame(data)
    
    # Printing the DataFrame
    print(df)
    
    # Output:
    #     Name  Age
    # 0    Tom   20
    # 1  Jerry   21

Flask

Flask is a lightweight Python web framework used to build web applications. It is known for its simplicity and flexibility, allowing developers to add only the components they need, making it a good choice for small to medium-sized applications. Flask does not come with the built-in features of a full-stack framework like Django, but it gives you the freedom to choose how to structure your app. 

# Installing Flask using pip
    pip install flask
    
    # Creating a simple Flask application
    from flask import Flask
    
    app = Flask(__name__)
    
    @app.route('/')
    def hello_world():
        return 'Hello, World!'
    
    # Running the Flask application
    if __name__ == '__main__':
        app.run()
